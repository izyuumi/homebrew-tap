# homebrew-tap

Homebrew tap for [@izyuumi](https://github.com/izyuumi) projects.

## Usage

```sh
brew tap izyuumi/tap
```

### Formulae

| Formula | Description |
|---------|-------------|
| `xcode-discord-rpc` | Discord Rich Presence client for Xcode |

```sh
brew install izyuumi/tap/xcode-discord-rpc
```

### Casks

| Cask | Description |
|------|-------------|
| `koe` | System-wide dictation for macOS |

```sh
brew install --cask izyuumi/tap/koe
```
